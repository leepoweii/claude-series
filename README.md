# Claude 系列教學

> 給自己開店、行銷帳務要自己來的人。
> 學會用 Claude 做日常工作，不需要懂技術。

YouTube 系列教學的配套素材 repo。每一集對應一個資料夾，裡面有講師腳本、學員教材、Demo 素材。

---

## 系列架構

| 系列 | 主題 | 對象 |
|---|---|---|
| **Ep 0** | 名詞解釋 | 所有人必看 |
| **Series 1** | Claude Chat | 完全沒用過 Claude |
| **Series 2** | Claude CoWork | 開始導入 AI 到日常工作流 |
| **Series 3（設計中）** | Claude Code | 完全解放 Claude 的潛力 |

---

## 資料夾結構

```
00-terms/             Ep 0：名詞解釋與課程大綱
01-Chat/              Series 1：Claude Chat
  Artifacts/            資料分析用 CSV 素材
02-Cowork/            Series 2：Claude CoWork
  Attachments/          半畝咖啡品牌素材、對帳單、供應商資料
    emails/             廠商對帳單 PDF
    半畝咖啡/           品牌 Context 素材
```

---

## 教學品牌：半畝咖啡

使用虛構品牌「半畝咖啡」作為教學案例。

> 金門金城鎮的日常咖啡廳，老闆阿凱從台北連鎖餐飲回鄉創業。
> 素材包含：品牌故事、菜單、月營運數據、供應商資訊、廠商對帳單。

所有人名、電話、地址均為虛構，僅供教學用途。
