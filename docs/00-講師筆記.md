## 教學眉角

### 第一章
- Chat vs CoWork demo 用**同一句話**問：「幫我想一個國慶連假的快閃活動」
- Chat 會給通用答案，CoWork 會根據半畝咖啡的菜單、客群、過往活動來回答
- 這個對比要做得夠戲劇化，學員才有感

### 第二章
- Step 2 查資料時，讓學員**各自問不同面向**（天氣、趨勢、在地活動），然後合在一起看
- Session Handoff 是亮點 — 先確認大家都有拿到 handoff prompt 再關對話
- Canva Connector 如果卡住，先跳過，用文字描述替代

### 第三章
- 對帳單 A 課前就要寄到學員 Gmail，**用你自己的 email 寄，發件人寫「島嶼咖啡供應」**
- 交貨記錄已經在第二章的 Context 裡了，不用另外載入
- 讓學員自己發現差異，不要直接給答案

### 第四章
- Skill 的 prompt 大概是：「把剛才的對帳流程變成一個 Skill，包含：從 Gmail 撈對帳單 → 比對 Context 裡的交貨記錄 → 產出差異報告」
- 對帳單 B **課中即時寄出**，製造「工作來了，一鍵處理」的臨場感
