# 錄影 Checklist — Claude CoWork 教學

> 按順序做，打勾就好。

---

## 一、錄影前準備（~15 min）

### 環境
- [ ] 關掉所有通知（勿擾模式）
- [ ] 關掉不相關的 app
- [ ] Chrome 只留乾淨的 Gmail tab
- [ ] 桌面整理乾淨（會被錄到）

### OBS 設定
- [ ] 螢幕錄製 + 麥克風
- [ ] 解析度 1920×1080
- [ ] 錄影格式 mkv（比 mp4 安全，不怕當機遺失）
- [ ] 試錄 10 秒，確認畫面 + 聲音正常
- [ ] 確認硬碟空間 > 10 GB

### Claude Desktop
- [ ] 建立新 CoWork Project：「半畝咖啡 — AI 協作營運」
- [ ] 載入以下 7 個檔案到 Knowledge：
  - `半畝咖啡-品牌故事.md`（品牌故事、定位）
  - `半畝咖啡-供應商.md`（廠商聯絡資訊）
  - `半畝咖啡-菜單.csv`
  - `半畝咖啡-月營運數據.csv`
  - `半畝咖啡-活動紀錄.csv`
  - `半畝咖啡-交貨記錄-島嶼咖啡供應-202609.csv`
  - `半畝咖啡-交貨記錄-金良食品行-202609.csv`
- [ ] 確認 Connectors 已授權（Google Drive、Gmail、Canva）
- [ ] **不要載入對帳單 PDF**

### Gmail
- [ ] 用自己 email 寄對帳單 A PDF（主旨：【島嶼咖啡供應】2026年9月對帳單）
- [ ] 確認收到了
- [ ] 對帳單 B PDF 準備好，放在桌面隨時可寄（第四章用）

---

## 二、開始錄影

- [ ] OBS 按下錄影
- [ ] 口述：「大家好，今天帶大家看 Claude CoWork 怎麼用在日常營運...」

---

## 三、錄影中 — 章節標記

> 每章開始時口述章節名，後製好切。

- [ ] **第一章**：Chat vs CoWork demo（同一句話對比）
- [ ] **第二章**：行銷企劃（查資料 → 企劃書 → Handoff → Canva）
- [ ] **休息**（可以暫停 OBS）
- [ ] **第三章**：對帳（Gmail Connector → 比對差異）
- [ ] **第四章**：Skills（建立 Skill → 寄對帳單 B → 一鍵對帳）
- [ ] **收尾**：回顧四個能力

---

## 四、錄影後

- [ ] OBS 停止錄影
- [ ] 確認檔案完整（可以播放）
- [ ] 備份到外接硬碟或 Google Drive
- [ ] 筆記：哪些地方需要重錄 / 卡住的地方
