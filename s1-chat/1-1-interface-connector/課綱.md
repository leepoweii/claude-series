## 介面導覽重點（2.5 min）

畫面走一遍，每個地方一句話帶過：

| 區域 | 說明 |
|---|---|
| 左側 sidebar | 對話歷史、Projects 入口 |
| 上方 dropdown | 切換模型（一般用預設就好）|
| 對話框左邊圖示 | Connectors / 搜尋工具 |
| 對話框右邊 | 上傳檔案 |
| 右側面板 | Artifacts 顯示在這裡 |
| Settings | Styles、Memory、Connectors 管理 |

> 📌 講師提示：不要每個都深入解釋，快速掃過，重點放在 Connectors 圖示

## Google Calendar Connector 設定 Demo（4 min）

**步驟：**
1. 點對話框左下角 → 搜尋工具 / Connectors
2. 找到 Google Calendar → 點 Connect
3. Google OAuth 授權流程（點同意）
4. 回到對話框，輸入測試問句

**測試問句：**
```
我這週還有哪些行程？哪天的空檔最多？
```

**預期效果：**
- AI 列出本週行程
- 指出哪天空白最多，建議可以排在那天

> 📌 講師提示：授權那個 Google 視窗可能彈出在後面，提醒觀眾注意

## Desktop Extensions 簡介（2 min）

**概念說明（不實作）：**

| | Connectors | Desktop Extensions |
|---|---|---|
| 資料在哪 | 雲端（Google、Canva 等）| 你的本機電腦 |
| 安裝方式 | Claude 介面直接連 | Claude Desktop → Settings → Extensions |
| 適合 | 雲端服務串接 | 讀寫本機檔案、私密資料 |

> 「如果你有一些資料放在自己電腦，不想上傳到外部——用 Desktop Extensions。下集會用到。」
