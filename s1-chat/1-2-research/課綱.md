# 1-2 — 研究 + 展示：用 AI 做市場分析

> Series 1：Claude Chat
> 時長：10 分鐘
> 前置：完成 1-1（有 Claude 帳號）

## 學完你能做到

- 用 Claude Web Search 查找並整合公開資料
- 把資料交給 Claude，產出一個互動式 Artifact 圖表
- 把 Artifact 分享連結給別人看

---

## 資料選擇（⚠️ 講師需挑選一個版本）

> data-finder agent 搜尋結果補充於此。以下為備選方案：

**【版本 A】台灣餐飲業登記資料（data.gov.tw）**
- 來源：政府公開資料，不需登入
- 內容：全台餐飲業登記數量，可按縣市分析
- Demo 情境：「我想在台灣開一間咖啡廳，先看看各縣市競爭有多激烈」
- Artifact：各縣市餐飲業數量長條圖 + 互動篩選

**【版本 B】咖啡廳銷售數據（Kaggle - Coffee Shop Sales）**
- 來源：Kaggle（需免費帳號下載）
- 內容：時段銷售、品項、金額
- Demo 情境：「分析一間咖啡廳的銷售數據，找出高峰時段和最賺品項」
- Artifact：每小時銷售折線圖 + 品項排名

**【版本 C】台灣家庭收支調查（data.gov.tw）**
- 來源：政府公開資料，不需登入
- 內容：各縣市平均可支配所得
- Demo 情境：「我想知道哪個縣市的消費力最強，選址參考」
- Artifact：各縣市收入比較圖

> 📌 **建議：版本 A 或 C**（不需 Kaggle 帳號，觀眾可以直接跟著做）
> 兩個都可以同時用，Demo 情境是「選址調查：競爭 vs 消費力」

## Demo 流程（版本 A + C 組合）

**Step 1 — 問 Claude 查資料（Web Search）**
```
我想在台灣開一間咖啡廳。幫我搜尋一下：
1. 台灣哪些縣市的咖啡廳競爭最激烈？
2. 各縣市的消費力差異大嗎？
請用 Web Search 找到實際數據。
```

**Step 2 — 載入 CSV（Data.gov.tw 下載的檔案）**
直接把 CSV 上傳到對話框，說：
```
這是台灣餐飲業登記資料，幫我用這個數據做一個互動式圖表，
讓我可以比較各縣市的餐飲業數量。
用 Artifact 做，要可以點選篩選。
```

**Step 3 — Artifact 產出**
- Claude 在右側面板生成互動圖表
- 示範點選互動（hover、篩選）

**Step 4 — 分享**
- 點 Artifact 右上角 Share
- 複製連結
- 說：「這個連結可以直接傳給朋友或貼在報告裡，對方不需要 Claude 帳號就能看」
