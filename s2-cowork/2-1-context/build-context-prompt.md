# 建立公司 Context 的引導 Prompt

> 下載這個檔案，把以下 prompt 貼給 Claude，讓 AI 引導你整理出自己公司的 Context。
> 適合在 CoWork Project 建立後使用。

---

## 使用方式

1. 在你的 CoWork Project 裡，開一個新對話
2. 把以下 prompt 完整貼進去
3. 照著 AI 問的問題回答
4. 最後讓 AI 把整理好的 Context 存成文件，上傳到 Project Knowledge

---

## Prompt

```
你是一位專業的品牌顧問，你的任務是幫我整理公司的 Context 文件，
讓 AI 助理能快速了解我的公司。

請一次問我一個問題，等我回答後再問下一個。
不要一次列出所有問題。
如果我的回答不清楚，請追問。
整理完成後，幫我把所有內容整理成一份結構化的 Markdown 文件。

從第一個問題開始：

---

請先告訴我，你的公司/店鋪叫什麼名字，主要做什麼？
```

---

## AI 會依序引導問你

AI 會根據你的回答，逐步問以下方向（不是全部，根據你的業態調整）：

**基本資訊**
- 公司名稱、成立時間、地點
- 主要產品或服務
- 定價區間

**客群**
- 主要客群是誰？年齡、職業、消費習慣
- 觀光客 vs 在地客的比例？
- 客人通常在什麼情況下找你？

**品牌**
- 一句話說你們公司是做什麼的
- 你們跟競爭對手差在哪裡？
- 品牌調性：溫暖/專業/活潑/文青？

**營運**
- 幾個人在跑？
- 旺季淡季的差異
- 目前最大的痛點是什麼？

**歷史與成就**
- 有什麼值得說的故事或里程碑？
- 做過什麼比較成功的活動或合作？

**供應商與合作夥伴**
- 主要的供應商有哪些？
- 有固定合作的廠商或品牌嗎？

---

## 完成後

AI 會整理出一份結構化的 Markdown 文件，你可以：
1. 複製到 Project Knowledge 上傳
2. 或直接讓 AI 在 Project 裡更新

這份 Context 之後可以隨時更新——有新活動、換了供應商、客群改變，
都可以開新對話告訴 AI，讓他幫你更新。
