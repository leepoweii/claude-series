# 2-2 — Context 實戰：從趨勢到企劃到海報

> Series 2：Claude CoWork（第二章節）
> 時長：10 分鐘
> 前置：完成 2-1（半畝咖啡 Project 已建立）

---

## 學完你能做到

- 用 Zero-shot 讓 AI 根據你的公司設計活動企劃
- 透過來回迭代修改企劃，貼近真實需求
- 用 Handoff Notes Skill 做工作交接（並預告 Skills 功能）
- 開新 Session 用 Handoff 接續工作，用 Canva Connector 產出海報

## Demo 流程

### Step 1 — 找趨勢新聞（1.5 min）

用 Web Search 或直接準備好一篇新聞（建議：節慶活動、飲食趨勢、旅遊熱點等）。
把文章貼進對話框。

> 📌 建議準備一篇：「國慶連假金門旅遊人數創新高」類型的新聞

---

### Step 2 — Zero-shot 企劃（3 min）

```
[貼上新聞內容]

幫我依照我們公司的狀態，設計一個類似的企劃。
```

AI 會根據 Project 裡的半畝咖啡資料，設計貼合的活動。

**迭代調整（示範 1-2 輪）：**
```
觀光客比例降低一點，主打在地客。活動時間改到下午。
```

```
預算控制在 5000 元以內，人力只有阿凱加一個兼職。
```

> 📌 重點：讓觀眾看到「AI 知道我們公司的狀況」才能迭代，這是 Context 的價值

---

### Step 3 — Handoff Notes Skill（2 min）

```
好，企劃定案了。現在用 Handoff Notes Skill，
把這個企劃整理成下一個 Session 可以接手的工作記錄。
目前狀態：企劃完成，接下來需要產出宣傳海報。
```

> 📌 提示給觀眾：「這個 Skill 是 Anthropic 官方提供的，不需要自己建立。2-3 集我們會學怎麼自己建 Skills。」

AI 產出結構化的 Handoff Notes，包含：
- 企劃摘要
- 已決定的事項
- 下一步任務
- 相關 context 摘要

**複製 Handoff Notes**

---

### Step 4 — 開新 Session + Canva 海報（2.5 min）

1. 開一個全新對話（或新 Session）
2. 貼上 Handoff Notes
3. 設定 Canva Connector（若未設定）：
   - 對話框 → Connectors → Canva → Connect → OAuth 授權
4. 輸入：
```
[貼上 Handoff Notes]

根據這個企劃，用 Canva 幫我設計一張 IG 宣傳貼文（1080x1080）。
風格：溫暖、日常、有咖啡廳感。
包含：活動名稱、時間、地點、一句 slogan。
```

5. Claude 透過 Canva Connector 建立設計稿，產出連結

---

## Canva Connector 設定說明

> 若學員未設定過 Canva Connector：

1. 對話框左下角 → 搜尋工具
2. 找到 Canva → Connect
3. Canva OAuth 授權（點同意）
4. 回到對話框，重新輸入指令

## 講師口語備忘

**開場：**
「企劃很花時間，因為你要同時想活動內容、想文案、想預算。AI 最擅長的就是這種需要整合很多資訊的事。今天我們示範：從一篇新聞，到一張海報，10 分鐘以內。」

**Zero-shot 後：**
「你注意到了嗎？他沒有給你一個通用的活動建議，他給你的是：金門石蚵、高粱拿鐵、觀光客、一個正職兩個兼職——這些都是他從 Context 裡知道的。這才是真正的 AI 協作。」

**迭代時：**
「這是 AI 跟 AI 工具最大的差別。AI 工具給你一個答案，AI 夥伴可以跟你來回調整，直到你滿意為止。」

**Handoff 時：**
「這個叫 Handoff Notes，是 Anthropic 官方提供的 Skill。它的用途是：我和 AI 討論到一半，要轉移到另一個任務，或者要讓另一個 Session 接手——這個筆記就是交接用的。下一集我們會學怎麼自己建 Skills。」

**Canva 海報後：**
「你看，他透過 Canva Connector 直接在 Canva 建立了一個設計——你可以點進去修改、下載，或直接發到 IG。從企劃到設計，中間我沒有打開任何其他 App。」
