> Series 2：Claude CoWork（第二章節）
> 時長：10 分鐘
> 前置：完成 2-1（半畝咖啡 Local Folder Project 已建立）

---

## 學完你能做到

- 用帶有完整指令的 prompt，讓 Context 發揮最大效果
- 連接 Canva Connector，讓 AI 直接在 Canva 建立設計稿
- 體驗「每次都要寫完整 prompt」的感覺（為 2-3 Skills 埋下對比）

---

## 核心概念：完整 Prompt × Context = 精準輸出

> Context 是 AI 的背景知識，Prompt 是你的指令。
> 這集刻意用完整、詳細的 prompt——讓觀眾感受到：有 Context 的時候，
> 完整的 prompt 能產出多精準的結果。
>
> 同時也讓你感受到：每次都要這樣寫，有點累。
> **這個「累」的感覺，是 2-3 Skills 的鋪墊。**

## Demo 流程

### Step 1 — 找趨勢新聞（1.5 min）

在 Claude CoWork（半畝咖啡 Project）用 Web Search，或直接貼一篇新聞：

> 建議新聞類型：「國慶連假金門旅客創新高」「台灣手工飲品市場趨勢」

把文章內容貼進對話框。

---

### Step 2 — 用完整 Prompt 做活動企劃（3.5 min）

**不要用 zero-shot 短句。** 用完整、結構化的 prompt：

```
根據以下這篇新聞，以及我們 Project 資料夾裡的半畝咖啡 Context（品牌故事、菜單、月營運數據、活動紀錄），幫我設計一個國慶連假快閃活動企劃。

請包含：
- 活動名稱和主題（要符合半畝咖啡的風格）
- 主打品項（從現有菜單選 2–3 項）
- 活動時間段安排（10/9–10/10）
- 預算估計（控制在 NT$5,000 以內）
- 人力配置（阿凱 + 1 名兼職）
- 預期效果（參考過去活動數據）

[貼上新聞內容]
```

**迭代一輪：**

```
觀光客比例調低一點，這次想主打在地熟客。
活動時間改到下午茶時段（14:00–17:00）。
加入一個可以在 IG 打卡的設計。
```

> 📌 讓觀眾注意：每次 AI 給的答案都用到了 Context 裡的資料——
> 阿凱的名字、高粱拿鐵、過去活動成效

---

### Step 3 — 連接 Canva Connector（1.5 min）

> 📌 若學員未設定過 Canva Connector：

1. 對話框左下角 → 搜尋工具（或 + 號）
2. 找到 Canva → Connect
3. Canva OAuth 授權（點同意）
4. 看到 Canva 工具出現在對話框

**說明：**
```
這是 Canva 官方的 Connector，不是外掛。
連一次之後，以後這個 Project 裡的對話都可以直接用。
```

---

### Step 4 — 用完整 Prompt 做 Canva 海報（2.5 min）

**同樣用完整指令：**

```
根據剛才定案的活動企劃，用 Canva 幫我設計一張 IG 宣傳貼文（1080×1080）。

設計要求：
- 風格：溫暖、質感、有咖啡廳日常感（不要太商業）
- 主色調：米白 + 深棕（配合半畝咖啡品牌色）
- 必須包含：活動名稱、時間（10/9–10/10 下午 14:00–17:00）、地點（金門半畝咖啡）
- 加入一句 slogan（根據活動主題發揮）
- 不要放價格，保持質感
```

Claude 透過 Canva Connector 建立設計稿，產出連結。

> 📌 點開連結，讓觀眾看到 Canva 裡真的有這份設計稿

---

### Step 5 — 預告 2-3（30 sec）

```
你有沒有注意到——這集我們每次都要寫很完整的 prompt。
企劃要說清楚格式、限制、風格；海報要說清楚尺寸、色調、內容。

下一集，我們把這些重複的指令存成 Skill。
以後同樣的事情，一個指令就觸發，不用每次重寫。
```

## 講師口語備忘

**開場：**
「上一集我們把 Context 放進去了。這集我們來用它——但有一件事要特別注意：Context 是背景知識，prompt 是你的指令。這兩個要配合好，AI 才能輸出精準的東西。」

**寫完整 prompt 時：**
「你看，我不是只說『幫我做一個活動』——我說清楚了時間、預算、人力、風格。為什麼要這樣？因為 AI 的 Context 已經知道你是誰，但他還是需要你告訴他你想要什麼。這兩個合在一起，才能出來你真正要的結果。」

**AI 回答後（強調 Context 的作用）：**
「你注意到了嗎？他提到阿凱、提到高粱拿鐵、提到你們過去辦過的活動成效——這些他都沒有問你，他是從 Project 資料夾裡讀來的。這就是 Context 的價值。」

**迭代時：**
「我改了一個方向，他馬上調整——而且調整的時候，他還是記得你的品牌風格和限制。這就是有 Context 的來回對話跟一般聊天的差別。」

**連接 Canva 時：**
「這是 Canva 官方的 Connector，Anthropic 跟 Canva 合作的。連一次就好，以後這個 Project 的對話裡都可以直接用。」

**海報出來後：**
「你看，他透過 Canva 直接建了一份設計稿——我們可以點進去改、下載、直接發 IG。從企劃到設計，我沒有打開任何其他 App。」

**結尾（鋪 2-3）：**
「但你有沒有發現——這集我們每次都要寫很長的 prompt。企劃要說格式、海報要說色調。如果下個月又要辦活動，我還要再寫一次嗎？下一集，我們把這個存成 Skill，一個指令觸發，不用重來。」
