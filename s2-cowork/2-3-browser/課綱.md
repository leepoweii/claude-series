## Demo 流程

### Step 1 — 安裝 Skills Generator（1.5 min）

> 📌 Skills Generator 是 Anthropic 官方提供的 Skill

1. Claude Desktop → Settings → Skills
2. 點「Browse Skills」或「Add from Anthropic」
3. 找到「Skills Creator / Skills Generator」→ 安裝

說明：
「這個 Skill 的用途是：幫你把手動做過的流程，存成一個可以重複使用的 Skill。等一下我們用手動跑完一次對帳，再讓他幫我們存起來。」

---

### Step 2 — 手動跑一次（3 min）

**連上 Gmail Connector（若未設定）：**
對話框 → Connectors → Gmail → Connect

**輸入：**
```
幫我找 Gmail 裡面，寄件人是「島嶼咖啡供應」的最新一封信，
把附件的 PDF 對帳單讀出來。
```

AI 透過 Gmail Connector 讀取附件 PDF 內容。

**比對：**
```
現在把這張對帳單，跟我們 Context 裡的 9 月叫貨記錄做比對，
找出有差異的地方。
用表格列出：品項、對帳單數量、我們記錄的數量、差異金額。
```

AI 產出差異報告（應找出：耶加雪菲多報 1 包 $850、多一筆濾掛包 $3,250）。

---

### Step 3 — 存成 Skill（2.5 min）

```
剛才這個對帳流程，幫我用 Skills Generator 存成一個 Skill。

Skill 名稱：廠商對帳
Skill 說明：從 Gmail 找廠商寄來的對帳單 PDF，比對 Context 裡的叫貨記錄，產出差異報告

請把完整的步驟和 prompt 都包含進去。
```

AI 使用 Skills Generator 建立 Skill，確認後儲存。

---

### Step 4 — 第二張對帳單一鍵跑（2 min）

**講師此時寄出對帳單 B（金良食品行）**

```
/廠商對帳
```

或直接叫出 Skill，指定廠商：
```
用廠商對帳 Skill，這次找金良食品行的對帳單。
```

AI 自動完成全流程，產出差異報告（應找出：奶油單價差 $160、多報雞腿排 $1,600）。

---

### Step 5 — Skills 共享說明（30 秒）

「這個 Skill 存在你的 CoWork 裡，你也可以匯出然後分享給同事，或者在 GitHub 上找別人做好的 Skill 直接用。不用每個人都重新建一遍。」

## 講師口語備忘

**開場：**
「老闆對帳，最討厭的是什麼？對帳單跟自己記的不一樣，但找不到差在哪裡。今天我們讓 AI 幫你做這件事，而且做完之後，存成一個 Skill，下次一個指令就跑完。」

**安裝 Skills Generator 時：**
「這個 Skills Generator 是 Anthropic 官方做的，不需要自己設定。它的用途很簡單：你手動做一件事，它幫你把這個流程存成可以重複用的 Skill。」

**AI 讀取 PDF 後：**
「你看，他讀到了 PDF 的內容。這個是透過 Gmail Connector——他沒有在你的帳號裡亂翻，他只讀了你指定的那封信的附件。」

**差異報告出來後：**
「找到了。這兩個地方有問題——耶加雪菲我們記錄是 5 包，他開單寫 6 包；還有一筆我們沒有訂的濾掛包。如果沒有 AI 幫你對，你可能直接付了。」

**存成 Skill 後：**
「好，流程存起來了。現在我們來試試看——講師剛剛寄了第二張對帳單進來，金良食品行的。我們一個指令直接跑。」

**一鍵完成後：**
「這就是 Skill 的威力。第一次手動，存起來，之後自動。而且你可以把這個 Skill 分享給同事，他們也可以直接用，不用重新建。」
