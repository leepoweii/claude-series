## Demo 情境說明

半畝咖啡收到一個工作坊報名邀請，你正在外面。用 Dispatch 說：

```
有一個 AI 工作坊的報名表，連結是
https://docs.google.com/forms/d/e/1FAIpQLScBetT-o6ExZNHB14H5w5Cy1_TGzw2XiSJZCrpVxCokDKJpyA/viewform

幫我根據半畝咖啡的 Context，把這份表單填好後提交。
不確定的欄位先空著，跟我確認再填。
```

**AI 能從 Context 自動填的欄位：**
- 報名人姓名 → 阿凱（品牌故事裡有）
- 公司名稱 → 半畝咖啡
- 聯絡電話 → 供應商聯絡資料裡有
- 電子信箱 → 同上
- 主要行業別 → 餐飲業
- 公司規模 → 2–5 人

**AI 需要問你確認的：**
- 最想解決的工作痛點（主觀，需要你填）
